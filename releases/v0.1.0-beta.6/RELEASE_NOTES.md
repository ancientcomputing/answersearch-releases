# AnswerSearch 0.1.0-beta.6

Unsigned beta installer for Windows 11 x64.

## Install notes

- This is an unsigned beta build, so Windows may show an unknown-publisher or SmartScreen warning.
- AnswerSearch runs locally and requires your own Anthropic API key.
- The SHA256 checksum is published beside the installer as `.sha256`.

## Changes

- Adds shared legal/support content generation for web, app, and installer surfaces.
- Shows the AI answer caution below the main question box.
- Installer license screen now uses generated AnswerSearch Terms of Use.
